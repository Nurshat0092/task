1) После запуска программы, ожидается инпут. Что бы закончить ввод данных, нужно нажать CTRL-D
2) Далее по колличеству ядер, запускаются горутины с ровно поделянными по количеству URL-аддресами
* Если количество URL = 0, то выдается соответствующая ошибка
  Если количество URL меньше количества ядер, то используются столько ядер, сколько URL-аддресов
3) Далее каждая горутина в цикле отправляет запросы по аддресам.
4) При завершении всех запросов выводится статистика, кол-во запросов на каждом ядре.
* При форсированном остановлении программы(CTRL-C), выводится статистика на данный момент.
